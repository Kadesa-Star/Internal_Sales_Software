About the Project

The Internal Sales Software was inspired by my experience working at a company that relied heavily on manual processes, leading to inefficiencies in stock and sales tracking. This project aims to provide a streamlined, automated solution.

This project is part of the Holberton School Portfolio Program. Learn more about the code on GitHub.
